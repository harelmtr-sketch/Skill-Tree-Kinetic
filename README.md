# Skill-Tree-Kinetic

## Manual edge wiring in `viewer.html`
1. Open `viewer.html`.
2. Click **Debug View: Off** to switch to a clean dark background without the tree image.
3. Click **Edge Edit: Off** to enable manual connection mode.
4. Click node A, then node B to add each edge.
5. Click **Node Move: Off** to enable dragging nodes to new positions.
6. Use **Clear Edges Only** to reset connections while keeping all node positions.
7. Click **Download skill_tree_graph.json** to save your final manual graph JSON.
